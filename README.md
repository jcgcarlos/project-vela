# Project Vela — Fintech User Behavior Analysis

> What drives high-value user behavior on a B2B fintech platform —
> and can we identify users at risk of disengaging before they do?

## Overview
Vela is a fictional B2B fintech platform. This project analyzes user 
behavior, feature adoption, and transaction patterns across 70,000+ 
transactions to surface insights that drive retention and growth.

## Business Questions
1. Do users who activate more features transact more frequently and 
   at higher value?
2. What separates power users from casual users?
3. Which users are showing early signs of disengagement?
